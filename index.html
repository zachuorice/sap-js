Super Audio Player An HTML5 audio player UI.

Welcome!

Super Audio Player is an jquery plugin for turning HTML into an interactive audio player!

See demo

Download

<ol id="tracklist" class="sap-playlist" style="display:none">
    <li>

        <audio class="sap-playlist-track-audio" src="ogg/gentlep.ogg">
        </audio>
        <div class="sap-playlist-track-frame">
            <h3>Learn More About This Track</h3>

    </li>
    <li>
